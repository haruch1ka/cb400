# Flip View Y — インストール方法

3Dビューポートを鏡像反転して、モデルの対称性やシルエットを確認するためのアドオン。

## インストール手順

1. Blender を起動する
2. `編集（Edit）> プリファレンス（Preferences）> アドオン（Add-ons）` を開く
3. 右上の `∨`（Blender 4.2以降）または `インストール（Install...）` ボタンから
   **「ディスクからインストール（Install from Disk...）」** を選択
4. このフォルダの `flip_view_y.py` を選択してインストール
5. 一覧に表示された **「Flip View Y」** のチェックボックスを ON にして有効化

> Blender 4.2 以降なら、`flip_view_y.py` を Blender のウィンドウに
> ドラッグ＆ドロップするだけでもインストールできる。

## 使い方

- 3Dビューポートのヘッダー `ビュー（View）` メニュー最下部に
  **「Flip View（左右反転）」「Flip View（上下反転）」** が追加される
- もしくは `F3` を押して「Flip View」で検索して実行

## 動作仕様

- 実行するとビューポートの表示が鏡像反転する。もう一度実行すると元に戻る
- 反転はビュー行列への一時的な操作のため、**視点を回転（オービット）すると
  自動的に元の表示に戻る**。一時的な対称性チェック向けの挙動
- テンキー `7`（トップビュー）で真上から見た状態で実行すると、
  トップビューを反転した状態で確認できる

## アンインストール

`プリファレンス > アドオン` で「Flip View Y」を検索し、
チェックを外す（無効化）か、`∨ > Uninstall` で削除する。
