# Phase 2: タイヤ周り（ホイール・タイヤ・ブレーキ）手順書

作成日: 2026-07-07

> **目標**: ブロッキングの円柱ホイールを、タイヤ・リム・スポーク・ハブ・ブレーキに分解してハードサーフェス化する。
> フロントを完成させてからリアに展開する（作り直しを1回で済ませるため）。

---

## 寸法リファレンス

| 項目 | フロント | リア |
|------|---------|------|
| タイヤ | 120/60ZR17 | 160/60ZR17 |
| タイヤ幅 | 120 mm | 160 mm |
| サイドウォール高（幅×扁平率） | 72 mm | 96 mm |
| タイヤ外径 | 約 576 mm | 約 624 mm |
| リム径（17インチ） | 431.8 mm | 431.8 mm |
| リム幅（目安） | 3.50インチ ≒ 89 mm | 4.50インチ ≒ 114 mm |
| ブレーキディスク | φ296 mm × **2枚** | φ240 mm × 1枚 |
| キャリパー | 2ポット | 1ポット |

> **注意**: [`cb400sf-specs.md`](../references/cb400sf-specs.md) のタイヤ外径「638mm」は誤記。計算値 576mm が正しい（17×25.4 + 2×120×0.60）。
> スポーク本数・断面形状は NC42 の実車写真（`blender/references/photos/`）で必ず確認してから着手する。

---

## Step 0: 準備

1. リファレンス写真でスポーク本数・リム断面・ディスク形状を確認する
   - 不足していれば `CB400SF NC42 wheel` / `front wheel closeup` で追加収集
2. `Wheels` コレクション内にサブコレクションを作る: `Wheel_Front` / `Wheel_Rear`
3. ブロッキングの円柱ホイールは**削除せず**、別レイヤーに退避してガイドとして使う
4. フロントホイール中心に **Empty（Plain Axes）** を置く → 回転体・Array の中心として全 Step で使い回す
   - 位置はブロッキング円柱の原点にスナップ（QuickSnap: `Ctrl+Shift+V`）

**完了条件**: ホイール中心 Empty が正しい位置（Z=タイヤ外径の半分）にある。

---

## Step 1: タイヤ本体（フロント）

回転体で作る。トレッドパターンはここでは作らない（方針は Step 8 参照）。

1. 側面ビューで Plane を追加 → 頂点を編集してタイヤの**断面プロファイル**を作る
   - 断面の外周: リム端（半径 216mm 付近）→ サイドウォール → トレッド面（半径 288mm）
   - タイヤはバイク用なので**トレッド面が丸い**（クルマと違い断面が半円状）。頂点 10〜14 個で表現
2. プロファイルに **Screw モディファイア** を追加
   - Axis: Y、Axis Object: Step 0 の中心 Empty
   - Steps: 48〜64（Subdivision を後がけするなら 32 でも可）
3. サイドウォールのリム接合部に段差（ビード部）を 1 段つける
4. Shade Smooth + Auto Smooth（またはウェイトノーマル）で確認

**完了条件**: 側面ビューでブロッキング円柱と外径が一致し、正面ビューで断面の丸みがバイクタイヤに見える。

---

## Step 2: リム

1. タイヤと同様に**リム断面プロファイル**を作成 → Screw
   - 外径 431.8mm、幅 89mm（フロント）
   - 断面: リムフランジ（タイヤを挟む爪）→ ドロップセンター（中央の凹み）
2. リム内側にスポーク接合面となる平面部を残しておく
3. エアバルブを Cylinder で 1 本追加（φ8mm 程度、リム内周に斜め配置）

**完了条件**: タイヤとリムの間に隙間・めり込みがない。

---

## Step 3: スポーク

**1本だけ作って放射コピー**する。本数は Step 0 で確認した実車写真に合わせる（NC42 はキャストホイール）。

### 3-1. Array 用の回転 Empty を作る

1. 中心 Empty（Step 0）と同じ位置に Empty をもう 1 個追加（`Shift+S` → Cursor to Selected → 追加）
2. この回転用 Empty を **Y 軸（ホイール軸）に 360÷スポーク本数 ° 回転**させる（例: 3本なら 120°、6本なら 60°）

### 3-2. スポーク 1 本をボックスモデリング

1. **スポークオブジェクトの原点をホイール中心に置く**（`Shift+S` でカーソルを中心 Empty へ → Set Origin to 3D Cursor）
   - 原点が中心からズレていると Array が螺旋状に崩れる。放射 Array 最大のハマりどころ
2. 正面ビュー（ホイール軸方向から見る向き）で Cube を追加し、リム内周〜ハブ外周に届く長さに伸ばす
3. Loop Cut で長さ方向に 3〜4 分割し、写真に合わせて絞り・湾曲をつける
   - キャストホイールはハブ側が太く、リム側へ向かって薄く広がるのが典型
4. 断面の彫り込み: 側面に Loop Cut を入れ、中央の面列を選択して押し込む（U 字/H 字断面）
5. Bevel モディファイアでエッジに鋳造らしい丸みをつける（幅 2〜3mm）

### 3-3. 放射 Array

1. Array モディファイアを追加
2. **Relative Offset をオフ**、**Object Offset をオン** → 3-1 の回転用 Empty を指定
3. Count = スポーク本数
4. リム・ハブとの接合部はめり込ませて OK（Boolean 不要）

### 別解: ホイール 1/n セグメント方式

スポークがリム・ハブと一体の鋳造形状に見える場合は、スポーク単体ではなく
**「リム＋スポーク＋ハブを含む 1/n 楔形」を作って Array + Merge で一周させる**方が接合部が綺麗。
Step 5 のディスクと同じ要領。どちらにするかは写真の一体感で判断する。

**完了条件**: 正面ビューで等間隔にスポークが並び、リム・ハブ両方に接続して見える。

## Step 4: ハブ・アクスル

1. ハブ本体: Cylinder（または断面 + Screw）で中央の筒を作る
   - フロントハブ幅は左右フォーク間に収まる幅（約 150mm）
2. アクスルシャフト: φ15mm 程度の Cylinder をホイール中心に貫通させる
3. 左右のカラー（スペーサー）を短い Cylinder で追加
4. ディスク取付面（左右に平らなフランジ、ボルト穴 5〜6 個は Phase 3 でよい）

**完了条件**: 側面ビューでハブがスポークの中心に収まり、フォーク位置と干渉しない。

---

## Step 5: ブレーキディスク（フロント: φ296 × 2枚）

ディスクは**円周方向に繰り返しパターン**なので、1/n セグメントを作って Array で閉じる。

1. 正面ビューで円盤の 1/6 セグメント（60°分）をモデリング
   - 外周リング（摺動面）+ 内周（ハブ取付部）+ つなぎのアーム
   - 摺動面の冷却穴（円形の穴）はこの段階で開けておく（LoopTools → Circle で整円化すると楽）
2. Array モディファイア（回転 Empty 方式、Count 6）+ Merge で一周させる
3. フローティングピン（外周と内周をつなぐボタン状部品、8〜10 個）を 1 個作って同じく放射 Array
4. 厚み: 4〜5mm（Solidify でも可）
5. **左右 2 枚**: Mirror モディファイア（ホイール中心 Empty を Mirror Object に）

**完了条件**: 正面ビューでディスクが一周つながっており、左右対称に 2 枚ある。

---

## Step 6: ブレーキキャリパー（2ポット）

1. 側面ビューで Cube からキャリパー本体をボックスモデリング
   - ディスクを跨ぐコの字型。サイズ目安: 縦 80 × 横 100 × 厚 60mm
   - Bevel でエッジを丸める（鋳造部品の丸み）
2. ブリードバルブ・バンジョーボルト（ブレーキホース接続部）を小 Cylinder で追加
3. キャリパーサポート（フォークへの取付ブラケット）を板状に作る
4. 位置: フォークの後方、ディスク外周に噛む位置
5. 反対側は Step 5 と同じ Mirror でコピー

**完了条件**: ディスクがキャリパーの隙間を通っており、フォークと自然につながって見える。

---

## Step 7: リアへ展開

フロント一式を複製して調整する。ゼロから作らない。

1. `Wheel_Front` コレクションごと複製 → `Wheel_Rear` へ
2. X 位置をリアホイール中心へ移動（ホイールベース 1,410mm）
3. 調整項目:
   - タイヤ: 幅 120 → **160mm**、外径 576 → **624mm**（断面プロファイルを編集して Screw が追従）
   - リム幅: 89 → **114mm**
   - ディスク: φ296 → **φ240、1枚のみ**（Mirror 削除、右側に配置）
   - キャリパー: 1ポットに小型化、スイングアーム下側に配置
4. リア固有パーツを追加:
   - **スプロケット**: ディスクと同じ 1/n セグメント + Array 方式（歯数 40 前後、左側）
   - **ハブダンパー部**: スプロケット取付側のハブを一回り太く

**完了条件**: 側面ビューで前後タイヤの外径差（576 vs 624mm）が出ており、リア左にスプロケット・右にディスクがある。

---

## Step 8: 仕上げ・検証

- [ ] 全オブジェクトに `Ctrl+A` → Apply Scale
- [ ] 命名を戦略に合わせる: `tire_front` / `wheel_front` / `brake_disc_front_L` など
- [ ] 各パーツ完成時の UV 展開（回転体は Follow Active Quads が楽）
- [ ] 側面ビューでリファレンス画像と重ねて輪郭確認
- [ ] ブロッキング用の円柱ホイールを削除

### トレッドパターンの方針（ここで決める）

| 方式 | 向き | 手間 |
|------|------|------|
| **ノーマルマップ（推奨）** | 引きのレンダリング中心 | Phase 4 のテクスチャ工程で対応。ジオメトリ不要 |
| ジオメトリ | タイヤのクローズアップを撮る | 溝を断面プロファイルに彫り込み + Screw の再利用 |

迷ったらノーマルマップにして Phase 4 へ先送りする（後からジオメトリ化も可能）。

---

## 作業中の注意

- **Screw / Array / Mirror はできるだけ最後まで非適用**のまま残す（断面の修正が全周に反映できる）
- Empty を動かすと回転体全体がズレる。中心 Empty は位置ロック（Transform の鍵アイコン）推奨
- ポリゴンが増えやすい工程なので、Screw の Steps は控えめ + Subdivision で調整
- こまめに保存。Step 完了ごとに別名バックアップ推奨
