# 足回りのモデリング方法（フォーク・スイングアーム・リアショック）

---

## フロントフォーク（テレスコピック正立フォーク）

### 構造

```
トップブリッジ
  │
  ├── アウターチューブ（太い、下側）  ← 固定
  └── インナーチューブ（細い、上側）  ← ストローク部分
```

### 作り方

1. Cylinder（φ 40mm）でインナーチューブを作る
2. 下端にアウターチューブ（φ 48mm）を追加、一部が重なる形
3. 左右2本を Y 軸方向に対称配置
4. **Mirror モディファイア**で左右対称を維持
5. トップブリッジ（ハンドル取付板）は薄い Cube で表現
   - 幅: 約 250mm、厚み: 20mm

### 傾き

キャスター角 **25.5°** で前傾させる（X軸回転）

### 寸法

| 部位 | 寸法 |
|------|------|
| インナーチューブ径 | φ 41mm |
| フォーク全長（目安） | 約 600mm |
| ストローク | 120mm |

---

## スイングアーム

1. 側面ビューでアーム形状を Curve で描く
2. Curve → Mesh 変換後、断面形状を押し出して角パイプ状に整える
3. リアアクスル取付部（右端）とピボット（左端）を別途作る
4. **Mirror モディファイア**で左右対称

---

## リアサスペンション（プロリンク モノショック）

1. Cylinder でダンパー本体（細長い円柱、φ 40mm × 長さ 250mm）
2. スプリングは Array + Screw で螺旋を作るか、Phase 3 まで省略
3. 取付ブラケットは薄い Cube で上下に追加
